# 1. 游戏简介
## 1.1. 背景
* 由于人类的破坏和栖息地的丧失等因素，地球上濒临灭绝生物的比例正在以惊人的速度增长。一些不法分子将动物关进了笼子里，一些濒临灭绝的动物正在等待你救援，情系野生动物，建设生态文明！
